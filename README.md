# 📚 Library Management System

A Python command-line application to manage a library's book inventory, handle book issuance, and track returns.

## 🚀 Features
* **Inventory Management:** Add new books to the library system and view a list of currently available books.
* **Issue & Return Tracking:** Issue books to users (tracking the issue date and allotted days using Python's `datetime` module) and process returns.
* **Interactive CLI:** Provides a text-based menu for users to interact with the system directly in the terminal.
* **In-Memory Storage:** Uses Python dictionaries to efficiently store and manage book records during runtime.

## 📋 Prerequisites
Ensure you have Python installed on your system. You can check your version by running:
`python --version` (Requires Python 3.x)
